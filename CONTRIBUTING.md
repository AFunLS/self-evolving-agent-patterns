# Contributing to Self-Evolving AI Agent Patterns

Thanks for your interest in contributing! Here's how you can help.

## Ways to Contribute

### 1. Share Your Anti-Patterns
Have you encountered a failure mode while building AI agents? Document it using our template:

```markdown
## Anti-Pattern: [Name]
- **What happened:** [Description]
- **Cost:** [Budget/time wasted]
- **Root cause:** [Why it happened]
- **Fix:** [How you solved it]
- **Prevention:** [How to detect it early]
```

### 2. Add Examples
Working code examples in `examples/` are always welcome. Requirements:
- Must be self-contained (runs without external dependencies beyond `anthropic`)
- Must include comments explaining the pattern
- Must follow the two-paradigm discipline (no string-matching on LLM output)

### 3. Improve Documentation
Found something unclear? Typos? Better explanations? PRs welcome.

## Guidelines

- **Be practical.** Every pattern should come from real experience, not theory.
- **Show the anti-pattern.** Explain what NOT to do and why, not just the correct approach.
- **Include metrics.** If a pattern improved something, share the numbers.

## Code of Conduct

Be respectful. Focus on the ideas, not the person. We're all here to build better AI agents.

## Questions?

Open an issue or reach out at [tutuoai.com](https://tutuoai.com).
